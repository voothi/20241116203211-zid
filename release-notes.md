# Release Notes

## [v1.0.0] - 2026-01-06

Initial release of the ZID generator utility.

### Added
- **Configuration System**: Added `config.ini` support for customizable ZID formats, prefixes, and suffixes.
- **English Translation**: All comments and documentation translated to English.
- **Test Suite**: Comprehensive unit tests added in `tests/` directory.
- **Clipboard Integration**: Automatically copies generated ZID to the system clipboard.
- **Workflow Support**: Documentation following RFC and release-specific workflows.
